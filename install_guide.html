🎾 건부니스 테니스 앱 설치 안내 🎾

홈 화면에 추가하고 전용 앱처럼 편리하게 사용해보세요!

🍎 아이폰 (Safari)
1
화면 하단의 '공유' 아이콘
(사각형에 화살표 ⬆️)을 누릅니다.
2
메뉴를 위로 스크롤하여
'홈 화면에 추가' (➕)를 선택합니다.
3
우측 상단의 '추가'를 누르면
바탕화면에 아이콘이 생성됩니다.
🤖 안드로이드 (Chrome)
1
화면 우측 상단의 '메뉴' 아이콘
(점 3개 ⋮)을 누릅니다.
2
메뉴 목록에서
'홈 화면에 추가' 또는 '앱 설치'를 선택합니다.
3
팝업창에서 '설치'를 누르면
바탕화면에 아이콘이 생성됩니다.
💡 바탕화면에 생성된 '건부니스' 아이콘을 눌러주시면 모바일 앱으로 실행됩니다!
